# CrossPad PC Simulator

Quick reference for using the CrossPad PC simulator.
This file is actually rendered in the "Help" app.

## Keyboard Capture

Click the keyboard icon on the left side
of the device body to cycle capture modes:

- OFF  - keyboard input disabled
- FOCUS - capture when window is focused
- GLOBAL - capture always (system-wide)

## Pad Keys

The keyboard maps to the 4x4 pad grid:

| 1 | 2 | 3 | 4 | Pads 13-16 |
| Q | W | E | R | Pads 9-12  |
| A | S | D | F | Pads 5-8   |
| Z | X | C | V | Pads 1-4   |

## Navigation

| Key       | Action                  |
| Escape    | Close app, go to launcher |
| Enter     | Confirm / click         |
| Space     | Volume overlay toggle   |
| Ctrl      | Volume overlay toggle   |
| PageUp    | Focus previous widget   |
| PageDown  | Focus next widget       |
| Arrows    | Navigate within widgets |

## Encoder

- Mouse wheel: rotate encoder knob
- Middle click: encoder button press

## Audio & MIDI

Click the jack connectors on the device
edges to select audio/MIDI devices.

- Left edge: Audio OUT 1, OUT 2
- Top edge: Audio IN 1, IN 2
- Right edge: MIDI OUT, MIDI IN
